Cafe24 Animatr - How to use




Scale

Child element scales in to the center by default.

<div class= "jaszanimatr__scale-in" >
<div> content </div>
</div>

Child element scales out from the center by default.

<div class= "jaszanimatr__scale-out" >
<div> content </div>
</div>

Change the scale direction by adding one of the classes from the following table.

<div class= "jaszanimatr__scale-in  jaszanimatr__direction--top" >
<div> content </div>
</div>


Animation Direction Class Name
Center .jaszanimatr__direction--center
Top .jaszanimatr__direction--top
Right .jaszanimatr__direction--right
Bottom .jaszanimatr__direction--bottom
Left .jaszanimatr__direction--left

Change the scale speed by adding one of the classes from the following table. Speed is normal by default.

<div class= "jaszanimatr__scale-in  jaszanimatr__speed--faster" >
<div> content </div>
</div>


Animation Speed Class Name Time
Normal .jaszanimatr__speed--normal 0.4s
Slow .jaszanimatr__speed--slow 0.6s
Slower .jaszanimatr__speed--slower 0.8s
Slowest .jaszanimatr__speed--slowest 1s
Fast .jaszanimatr__speed--fast 0.3s
Faster .jaszanimatr__speed--faster 0.2s
Fastest .jaszanimatr__speed--fastest 0.1s

Add timing function by adding one of the classes from the following table. Timing function is cubic by default

<div class= "jaszanimatr__scale-in  .jaszanimatr__timing--ease-in" >
<div> content </div>
</div>


Timing Function Class Name
Ease .jaszanimatr__timing--ease
Ease In .jaszanimatr__timing--ease-in
Ease Out .jaszanimatr__timing--ease-out
Ease In Out .jaszanimatr__timing--ease-in-out
Linear .jaszanimatr__timing--linear
Cubic .jaszanimatr__timing--cubic

Change both direction and speed.

<div class= "jaszanimatr__scale-in  jaszanimatr__direction--top  jaszanimatr__speed--faster" >
<div> content </div>
</div>

Change both direction and timing.

<div class= "jaszanimatr__scale-in  jaszanimatr__direction--top  .jaszanimatr__timing--ease-out" >
<div> content </div>
</div>

Change direction, timing and speed.

<div class= "jaszanimatr__scale-in  jaszanimatr__direction--top  .jaszanimatr__timing--ease-out  .jaszanimatr__speed--slowest" >
<div> content </div>
</div>

Customize animation speed by changing the animation duration value.

.jaszanimatr__scale-in {
    animation-duration : 3s;
}

Customize animation delay by changing the animation delay value.

.jaszanimatr__scale-in {
    animation-delay : 2s;
}
